# Introduction to SwiftRequest

SwiftRequest is a lightweight, type-safe HTTP client for Swift, streamlining the construction and execution of HTTP requests. It abstracts away the repetitive boilerplate code that's typically associated with setting up HTTP requests in Swift.

SwiftRequest utilizes macros introduced in Swift 5.9, which can be associated with specific declarations to enhance and extend their functionality.

## Quick example

Here's how you define a service to fetch quotes:

```swift
@Service(resource: "quotes")
protocol QuoteService {
    @GET("random")
    func getRandomQuotes(@QueryParam limit: Int?) async throws -> [Quote]
    
    @GET("{id}")
    func getQuote(@Path by id: String) async throws -> Quote
}
```

To make a request, create an instance of the service and call the method:

```swift
let service = QuoteServiceImpl(baseURL: "https://api.quotable.io")
let quotes = try await service.getRandomQuotes(limit: 5)
let quote = try await service.getQuote(by: "69Ldsxcdm-")
```

## Core capabilities

- **Full HTTP method support**: GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS
- **Rich parameter handling**: Path parameters, query params, headers, body, and form fields
- **Automatic JSON encoding/decoding**: Seamless Codable integration
- **Custom executors**: Swap networking backends (URLSession, Alamofire, etc.)
- **Swift 5.9+ macros**: Built on the modern Swift macro system
