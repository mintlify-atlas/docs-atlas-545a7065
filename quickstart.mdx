# Quickstart

This guide will walk you through creating your first HTTP service with SwiftRequest. We'll build a quote service that fetches random quotes from an API.

## Overview

By the end of this guide, you'll have a working service that:
- Fetches random quotes from an API
- Retrieves a specific quote by ID
- Uses type-safe Swift code with no manual URL construction

## Step-by-step tutorial

<Steps>
  <Step title="Define your data model">
    First, create a `Codable` struct to represent the quote response:

    ```swift
    struct Quote: Codable {
        let id: String
        let content: String
        let author: String
        let tags: [String]
        
        enum CodingKeys: String, CodingKey {
            case id = "_id"
            case content
            case author
            case tags
        }
    }
    ```

    This maps to the JSON response from the quotable API.
  </Step>

  <Step title="Define your service protocol">
    Create a protocol with the `@Service` macro and define your endpoints:

    ```swift
    import SwiftRequest

    @Service(resource: "quotes")
    protocol QuoteService {
        @GET("random")
        func getRandomQuotes(@QueryParam limit: Int?) async throws -> [Quote]
        
        @GET("{id}")
        func getQuote(@Path by id: String) async throws -> Quote
    }
    ```

    <Note>
      The `@Service` macro generates an implementation class named `QuoteServiceImpl` that you'll use to make requests.
    </Note>

    Let's break down what's happening:
    - `@Service(resource: "quotes")` - Sets the base resource path to `/quotes`
    - `@GET("random")` - Defines a GET request to `/quotes/random`
    - `@QueryParam limit` - Adds an optional query parameter `?limit=N`
    - `@GET("{id}")` - Defines a GET request with a path parameter
    - `@Path by id` - Replaces `{id}` in the URL with the provided value
  </Step>

  <Step title="Create a service instance">
    Instantiate the generated service implementation with your API base URL:

    ```swift
    let service = QuoteServiceImpl(baseURL: "https://api.quotable.io")
    ```

    The base URL is combined with the resource path and endpoint paths to form complete URLs:
    - `getRandomQuotes(limit: 3)` → `https://api.quotable.io/quotes/random?limit=3`
    - `getQuote(by: "69Ldsxcdm-")` → `https://api.quotable.io/quotes/69Ldsxcdm-`
  </Step>

  <Step title="Make your first request">
    Call the service methods using async/await:

    ```swift
    do {
        // Fetch 3 random quotes
        let quotes = try await service.getRandomQuotes(limit: 3)
        print(quotes)
        
        // Fetch a specific quote by ID
        let quote = try await service.getQuote(by: "69Ldsxcdm-")
        print(quote)
    } catch {
        print("Error: \(error)")
    }
    ```
  </Step>
</Steps>

## Complete example

Here's the complete working example:

```swift
import Foundation
import SwiftRequest

// 1. Define the data model
struct Quote: Codable {
    let id: String
    let content: String
    let author: String
    let tags: [String]
    
    enum CodingKeys: String, CodingKey {
        case id = "_id"
        case content
        case author
        case tags
    }
}

// 2. Define the service
@Service(resource: "quotes")
protocol QuoteService {
    @GET("random")
    func getRandomQuotes(@QueryParam limit: Int?) async throws -> [Quote]
    
    @GET("{id}")
    func getQuote(@Path by id: String) async throws -> Quote
}

// 3. Create service instance
let service = QuoteServiceImpl(baseURL: "https://api.quotable.io")

// 4. Make requests
do {
    let quotes = try await service.getRandomQuotes(limit: 3)
    print(quotes)
    
    let quote = try await service.getQuote(by: "69Ldsxcdm-")
    print(quote)
} catch {
    print(error)
}
```

## Custom service executor

<Note>
  By default, SwiftRequest uses `URLSessionServiceExecutor`. You can provide a custom executor or use a different networking backend.
</Note>

You can customize the networking backend by passing a custom executor:

```swift
import SwiftRequest

let executor: any ServiceExecutor = URLSessionServiceExecutor()
let service = QuoteServiceImpl(
    baseURL: "https://api.quotable.io",
    executor: executor
)
```

SwiftRequest also supports using a custom `URLSession`:

```swift
let session = URLSession(configuration: .default)
let service = QuoteServiceImpl(
    baseURL: "https://api.quotable.io",
    session: session
)
```
